Zed is working through Cryptopals Challenge

Zed is working through https://cryptopals.com/ to learn how to actually use rust.

  • Challenges are solved by writing and running tests in relevant modules. They’re not solved or addressed directly. For example, you can’t find a crate::challenges::set1::challenge3 module that directly addresses https://cryptopals.com/sets/1/challenges/3.
  • That’s because the Cryptopals challenges build upon each other to form a comprehensive cryptographic toolkit.
  • Except: Some challenges that result in one-off code are addressed direcctly in crate::challenges.
